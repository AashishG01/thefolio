# **App Name**: MERN Folio

## Core Features:

- Smooth Scrolling Navbar: A sticky navbar with blurred background and smooth scrolling to sections.
- Dynamic Hero Section: A hero section with a typed-text animation and social media links.
- Interactive Project Cards: Display projects in a grid, each card showing title, description, tech stack, and links.
- Animated Page Transitions: Use Framer Motion for smooth transitions between different pages or sections.
- Contact Form Submission: Submit the contact form data to the backend and display a success notification upon successful submission using nodemailer.
- MongoDB Data Fetching: Fetch and display project and experience data from MongoDB.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) for a modern, professional feel.
- Background color: Light gray (#F9FAFB), almost white, to create a clean and spacious canvas.
- Accent color: Vibrant purple (#A855F7) to add flair and highlight key interactive elements.
- Headline Font: 'Space Grotesk' sans-serif for headlines, lending a techy and modern aesthetic.
- Body Font: 'Inter' sans-serif for the body, guaranteeing readability with its neutral and objective design.
- Use a consistent set of icons from 'react-icons' to represent skills and social media links.
- Implement a glassmorphism effect for the Navbar and project cards for a modern aesthetic.
- Subtle fade-in animations on scroll and hover effects for interactivity.
